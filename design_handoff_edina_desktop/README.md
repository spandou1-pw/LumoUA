# Handoff: Edina — Desktop/Web Social Network UI

## Overview
Дизайн десктопной/веб-версии украинской социальной сети **Edina**: вход/регистрация, лента постов со сторис, чаты в стиле Telegram, поиск, уведомления, профиль и настройки. Светлая и тёмная темы, акцентный голубой цвет.

## About the Design Files
Файл `Edina Desktop.dc.html` (+ `support.js`) в этом пакете — **дизайн-референс, созданный в HTML**. Это прототип, показывающий целевой вид и поведение, а НЕ production-код для прямого копирования. Задача: **воссоздать эти экраны в существующем окружении целевого сайта/приложения** (React, Vue, Svelte, нативный стек и т.д.), используя его паттерны и библиотеки. Если окружения ещё нет — выбрать подходящий фреймворк и реализовать дизайн в нём. Прототип открывается в браузере (держите `support.js` рядом) — используйте его как живой образец и DevTools для точных значений.

## Fidelity
**High-fidelity (hifi)** — финальные цвета, типографика, отступы и интерактивные состояния. UI следует воссоздавать пиксель-в-пиксель, но средствами целевой кодовой базы.

## Design Tokens
Темы реализованы через CSS-переменные. Акцент по умолчанию `#3390EC` (Telegram blue); пользователь выбрал тёмную тему как основную.

Светлая тема:
- `--bg: #f0f2f5` (фон страницы), `--panel: #ffffff` (карточки/панели), `--panel2: #f2f4f6` (инпуты, вторичные поверхности)
- `--border: #e4e7ea`, `--text: #111418`, `--text2: #75808c`, `--hover: #f4f6f8`
- `--active: color-mix(in srgb, var(--accent) 12%, transparent)` — фон активного пункта навигации
- `--bubble-in: #ffffff`, `--bubble-out: color-mix(in srgb, var(--accent) 16%, #ffffff)`
- `--shadow: 0 1px 2px rgba(16,24,40,.06)`

Тёмная тема (data-theme="dark"):
- `--bg: #0e1621`, `--panel: #17212b`, `--panel2: #242f3d`, `--border: #0b131c`
- `--text: #f1f4f7`, `--text2: #8a97a4`, `--hover: #1e2a36`
- `--active: color-mix(in srgb, var(--accent) 20%, transparent)`
- `--bubble-in: #182533`, `--bubble-out: #2b5278`
- `--shadow: 0 1px 3px rgba(0,0,0,.4)`

Прочие цвета: онлайн-индикатор `#31b545`, лайк/danger `#e0556f`.
Градиенты аватаров (135deg): pink `#e8879b→#d5586f`, green `#8bd17c→#5aa54c`, blue `#71b7ea→#3f7fc4`, violet `#b0a3ee→#8672d8`, rose `#f191bd→#d9569a`, teal `#7fd4d6→#48a8ab`, orange `#fbc27a→#eb8c3f`.

Типографика: системный стек `-apple-system, 'SF Pro Text', 'Segoe UI', system-ui, Roboto, sans-serif`; базовый размер 14px. Заголовки экранов 21px/700, имена 14.5px/600, вторичный текст 12–13.5px цветом `--text2`.

Радиусы: карточки 16px, строки/кнопки 10–12px, инпуты-пилюли 18–20px, аватары 50%.
Скроллбар: 6px, `rgba(128,128,140,.35)`.

## Interactions & Behavior
- Навигация: клик по рейлу переключает экраны (SPA, без перезагрузки).
- Чаты: выбор чата подсвечивает строку и меняет диалог; отправка (Enter или кнопка) добавляет исходящее сообщение с текущим временем и скроллит вниз.
- Лайк: тоггл, счётчик ±1, сердце заливается `#e0556f`.
- «Стежити» ↔ «Читаєте»: тоггл заливка/outline.
- Тема: тоггл в рейле и выбор в настройках; мгновенное переключение CSS-переменных (transition .15s на интерактивных элементах).
- Hover: пункты списка/строки — `--hover`; primary-кнопки — `filter: brightness(1.08)`.
- Переходы: все micro-transitions ~0.15s ease; появление сообщений fadeUp .18s.

## State Management
- `screen` (login | feed | chats | search | notifs | profile | settings)
- `theme` (light | dark; предпочтение пользователя — dark по умолчанию), `authMode` (in | up)
- `activeChat`, `draft`, отправленные сообщения по чатам
- `liked` (по постам), `following` (по аккаунтам), `profileTab`, тумблеры настроек
- Данные (чаты, посты, уведомления) в прототипе захардкожены — в реальном приложении приходят с API.

## Assets
- Иконки: инлайн-SVG (stroke 1.7–1.9, round caps, 18–23px) в стиле Feather/SF Symbols — в целевом проекте можно взять lucide/feather или SF Symbols.
- Аватары и фото: плейсхолдеры (градиенты + инициалы). Заменить на реальные изображения пользователей/постов.
- Логотип: буква «Є» на accent-фоне — временный, заменить на реальный логотип Edina.
